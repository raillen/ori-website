## Why Ori?

Ori prioritizes **readable code** with mandatory type annotations, `result` / `optional` for errors, and a native Cranelift backend. The standard library spans **Layer 1** (Rust runtime) and **Layer 2** (`.orl` wrappers).

## Quick example

```ori
namespace app.main

import ori.io as io

func main()
    const label: string = if true then "Ori" else "world"
    io.println("Hello, " + label + "!")
end
```
