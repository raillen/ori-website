## Requirements

- **Rust 1.95+** (to build the compiler from source)
- **Windows MSVC**, **Linux GNU**, or **macOS** for the native backend
- Packaged runtime under `runtime/<triple>/` (or build with `cargo build -p ori-runtime`)

## Build from source

```bash
git clone https://github.com/ori-lang/ori-lang.git
cd ori-lang
cargo build -p ori-driver
cargo run -p ori-driver -- check examples/hello_world.orl
cargo run -p ori-driver -- run examples/hello_world.orl
```

## Your first file

Create `main.orl`:

```ori
namespace app.main

import ori.io as io

func main()
    io.println("Hello, Ori!")
end
```

```bash
cargo run -p ori-driver -- check main.orl
cargo run -p ori-driver -- run main.orl
```

## Environment checks

```bash
cargo run -p ori-driver -- doctor
```

Verifies stdlib root, native runtime, linker strategy, and JIT availability.
