Lazy evaluation primitives. Defers computation until the value is first accessed.

### `force`

```ori
(arg0: lazy<_#0>) -> _#0
```

*Layer 1 · runtime* · **Aliases:** `lazy.force`

Forces evaluation of the lazy value and returns the computed result.

### `is_consumed`

```ori
(arg0: lazy<_#0>) -> bool
```

*Layer 1 · runtime* · **Aliases:** `lazy.is_consumed`

Returns true if the lazy value has already been evaluated.

### `once`

```ori
(arg0: func() -> _#0) -> lazy<_#0>
```

*Layer 1 · runtime* · **Aliases:** `lazy.once`

Creates a lazy value that computes its result on first access and caches it.
