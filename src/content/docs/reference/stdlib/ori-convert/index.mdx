Type conversion primitives between basic types (bool, float, string, int).

### `bool_to_string`

```ori
(arg0: bool) -> string
```

*Layer 1 · runtime* · **Aliases:** `bool_to_string`

Converts a boolean to its string representation ("true" or "false").

### `float_to_string`

```ori
(arg0: float) -> string
```

*Layer 1 · runtime* · **Aliases:** `float_to_string`

Converts a float to its string representation.

### `string_to_float`

```ori
(arg0: string) -> optional<float>
```

*Layer 1 · runtime* · **Aliases:** `string_to_float`

Parses a string as a floating-point number.

### `string_to_int`

```ori
(arg0: string) -> optional<int>
```

*Layer 1 · runtime* · **Aliases:** `string_to_int`

Parses a string as an integer.
