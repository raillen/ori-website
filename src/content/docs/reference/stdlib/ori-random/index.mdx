### `float`

```ori
(arg0: float, arg1: float) -> float
```

*Layer 1 · runtime* · **Aliases:** `random.float`

Generates a pseudorandom floating-point number between 0.0 and 1.0.

* **Applicability:** Useful for generating percentages, probability weights, random noise multipliers, and easing variations.

```ori
// Example
import ori.random as rnd
import ori.io as io

func main()
    const weight = rnd.float()
    io.println(string(weight)) // Prints random float between 0.0 and 1.0
end
```
