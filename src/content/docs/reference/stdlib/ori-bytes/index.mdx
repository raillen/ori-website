Raw byte buffers utilities. Allows operations such as hex conversions, slice manipulation, and list conversions.

### `concat`

```ori
(arg0: bytes, arg1: bytes) -> bytes
```

*Layer 1 · runtime* · **Aliases:** `bytes.concat`

Concatenates two strings into one.

### `decode_utf8`

```ori
(arg0: bytes) -> result<string, string>
```

*Layer 1 · runtime* · **Aliases:** `bytes.decode_utf8`

Decodes the byte buffer as a UTF-8 string.

### `from_hex`

```ori
(arg0: string) -> result<bytes, string>
```

*Layer 1 · runtime* · **Aliases:** `bytes.from_hex`

Parses a hexadecimal string and returns the corresponding raw bytes buffer.

* **Applicability:** Useful for decoding hex-encoded files, processing public-key certificates, or parsing network packets represented as hex.

```ori
// Example
import ori.bytes as bytes
import ori.io as io

func main()
    const hex_str = "48656c6c6f"
    const data = bytes.from_hex(hex_str)
    io.println(string(data)) // Prints "Hello"
end
```

### `from_list`

```ori
(arg0: list<int>) -> bytes
```

*Layer 1 · runtime* · **Aliases:** `bytes.from_list`

Creates a new collection initialized with the elements from the given list.

### `get`

```ori
(arg0: bytes, arg1: int) -> u8
```

*Layer 1 · runtime* · **Aliases:** `bytes.get`

Returns the element at the specified index without removing it.

### `len`

```ori
(arg0: bytes) -> int
```

*Layer 1 · runtime* · **Aliases:** `bytes.len`

Returns the number of elements in the collection.

### `slice`

```ori
(arg0: bytes, arg1: int, arg2: int) -> bytes
```

*Layer 1 · runtime* · **Aliases:** `bytes.slice`

Returns a new collection containing elements from the specified start to end index.

### `to_hex`

```ori
(arg0: bytes) -> string
```

*Layer 1 · runtime* · **Aliases:** `bytes.to_hex`

Converts a raw bytes buffer into its hexadecimal string representation.

* **Applicability:** Widely used in cryptographic hashing visualizations, generating checksum strings, and formatting binary protocol payloads for debugging.

```ori
// Example
import ori.bytes as bytes
import ori.io as io

func main()
    const data = bytes.from_list([0, 15, 255])
    io.println(bytes.to_hex(data)) // Prints "000fff"
end
```

### `to_list`

```ori
(arg0: bytes) -> list<int>
```

*Layer 1 · runtime* · **Aliases:** `bytes.to_list`
