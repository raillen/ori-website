Binary heap (priority queue) data structure. Supports efficient insertion and extraction of the highest-priority element.

### `clear`

```ori
(arg0: heap.Heap<_#0>) -> void
```

*Layer 1 · runtime* · **Aliases:** `heap.clear`

Removes all elements from the collection, resetting it to empty.

### `clone`

```ori
(arg0: heap.Heap<_#0>) -> heap.Heap<_#0>
```

*Layer 1 · runtime* · **Aliases:** `heap.clone`

Returns a deep copy of the collection, independent of the original.

### `from_list`

```ori
(arg0: list<_#0>) -> heap.Heap<_#0>
```

*Layer 1 · runtime* · **Aliases:** `heap.from_list`

Creates a new collection initialized with the elements from the given list.

### `into_sorted_list`

```ori
(arg0: heap.Heap<_#0>) -> list<_#0>
```

*Layer 1 · runtime* · **Aliases:** `heap.into_sorted_list`

Drains the heap and returns all elements as a sorted list.

### `is_empty`

```ori
(arg0: heap.Heap<_#0>) -> bool
```

*Layer 1 · runtime* · **Aliases:** `heap.is_empty`

### `len`

```ori
(arg0: heap.Heap<_#0>) -> int
```

*Layer 1 · runtime* · **Aliases:** `heap.len`
