Double-ended queue (deque) data structure. Supports efficient insertion and removal from both ends.

### `back`

```ori
(arg0: deque.Deque<_#0>) -> optional<_#0>
```

*Layer 1 · runtime* · **Aliases:** `deque.back`

Returns the element at the back without removing it.

### `clear`

```ori
(arg0: deque.Deque<_#0>) -> void
```

*Layer 1 · runtime* · **Aliases:** `deque.clear`
