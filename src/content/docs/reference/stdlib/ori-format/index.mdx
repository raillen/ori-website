Value formatting primitives for rendering numbers, dates, bytes, and other types as human-readable strings.

### `binary`

```ori
(arg0: int) -> string
```

*Layer 1 · runtime* · **Aliases:** `format.binary`

Formats a number as a binary (base-2) string.

### `bytes_size`

```ori
(arg0: int, arg1: string) -> string
```

*Layer 1 · runtime* · **Aliases:** `format.bytes_size`

Formats a byte count as a human-readable size string (e.g., "1.5 MB").

### `date`

```ori
(arg0: int, arg1: string) -> string
```

*Layer 1 · runtime* · **Aliases:** `format.date`

Formats a timestamp as a date string.

### `datetime`

```ori
(arg0: int, arg1: string, arg2: string) -> string
```

*Layer 1 · runtime* · **Aliases:** `format.datetime`

Formats a timestamp as a date-time string.

### `hex`

```ori
(arg0: int) -> string
```

*Layer 1 · runtime* · **Aliases:** `format.hex`

Formats a number as a hexadecimal string.

### `number`

```ori
(arg0: float, arg1: int) -> string
```

*Layer 1 · runtime* · **Aliases:** `format.number`

Formats a number with locale-aware separators.

### `percent`

```ori
(arg0: float, arg1: int) -> string
```

*Layer 1 · runtime* · **Aliases:** `format.percent`

Formats a float as a percentage string.
