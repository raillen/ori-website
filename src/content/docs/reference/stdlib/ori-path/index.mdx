### `join`

```ori
func join(parts: list<string>) -> string
```

*Layer 2 · .orl* · **Source:** `path.orl`

Combines list segments into a unified file system path, automatically resolving separator slashes.

* **Applicability:** Used to dynamically build local paths cross-platform without worrying about '/' vs '\' differences.

```ori
// Example
import ori.path as path
import ori.io as io

func main()
    const target = path.join("docs", "spec", "errors.md")
    io.println(target) // Prints "docs/spec/errors.md" (or system equivalent)
end
```
