Atomic operations for lock-free concurrent value access. Provides atomic load, store, and add operations.

### `add`

```ori
(arg0: atomic.AtomicInt, arg1: int) -> int
```

*Layer 1 · runtime* · **Aliases:** `atomic.add`

Adds an element to the set. Returns true if the element was newly inserted.

### `load`

```ori
(arg0: atomic.AtomicInt) -> int
```

*Layer 1 · runtime* · **Aliases:** `atomic.load`

Atomically reads the current value.

### `new`

```ori
(arg0: int) -> atomic.AtomicInt
```

*Layer 1 · runtime* · **Aliases:** `atomic.new`

Creates a new empty instance of the atomic collection.

### `store`

```ori
(arg0: atomic.AtomicInt, arg1: int) -> void
```

*Layer 1 · runtime* · **Aliases:** `atomic.store`

Atomically replaces the stored value.
