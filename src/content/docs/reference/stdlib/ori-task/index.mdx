Async task management. Provides spawning, joining, cancellation, and coordination of asynchronous computations.

### `associate`

```ori
(arg0: task.CancelToken, arg1: future<_#0>) -> void
```

*Layer 1 · runtime* · **Aliases:** `task.associate`

Associates the task with a cancellation token for cooperative cancellation.

### `block_on`

```ori
(arg0: future<_#0>) -> _#0
```

*Layer 1 · runtime* · **Aliases:** `task.block_on`

Blocks the current thread until the given async operation completes.

### `cancel`

```ori
(arg0: task.CancelToken) -> void
```

*Layer 1 · runtime* · **Aliases:** `task.cancel`

Cancels the running task.

### `create_token`

```ori
() -> task.CancelToken
```

*Layer 1 · runtime* · **Aliases:** `task.create_token`

Creates a cancellation token that can be shared across tasks.

### `detach`

```ori
(arg0: task.Job<_#0>) -> void
```

*Layer 1 · runtime* · **Aliases:** `task.detach`

Detaches the task so it runs independently without a join handle.

### `is_cancelled`

```ori
(arg0: task.CancelToken) -> bool
```

*Layer 1 · runtime* · **Aliases:** `task.is_cancelled`

Returns true if the task has been cancelled.

### `join`

```ori
(arg0: task.Job<_#0>) -> result<_#0, task.JoinError>
```

*Layer 1 · runtime* · **Aliases:** `task.join`

Joins a list of strings with the given separator.

### `sleep`

```ori
(arg0: int) -> future<void>
```

*Layer 1 · runtime* · **Aliases:** `task.sleep`

Suspends the current task for the specified duration in milliseconds.

### `spawn`

```ori
(arg0: func() -> _#0) -> task.Job<_#0>
```

*Layer 1 · runtime* · **Aliases:** `task.spawn`

Spawns a new async task that runs concurrently.
