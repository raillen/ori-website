FIFO queue data structure. Supports enqueue, dequeue, and peek operations.

### `clear`

```ori
(arg0: queue.Queue<_#0>) -> void
```

*Layer 1 · runtime* · **Aliases:** `queue.clear`

Removes all elements from the collection, resetting it to empty.

### `clone`

```ori
(arg0: queue.Queue<_#0>) -> queue.Queue<_#0>
```

*Layer 1 · runtime* · **Aliases:** `queue.clone`

Returns a deep copy of the collection, independent of the original.

### `dequeue`

```ori
(arg0: queue.Queue<_#0>) -> optional<_#0>
```

*Layer 1 · runtime* · **Aliases:** `queue.dequeue`

Removes and returns the front element from the queue.

### `enqueue`

```ori
(arg0: queue.Queue<_#0>, arg1: _#0) -> void
```

*Layer 1 · runtime* · **Aliases:** `queue.enqueue`

Adds an element to the back of the queue.

### `is_empty`

```ori
(arg0: queue.Queue<_#0>) -> bool
```

*Layer 1 · runtime* · **Aliases:** `queue.is_empty`

Returns true if the collection contains no elements.

### `len`

```ori
(arg0: queue.Queue<_#0>) -> int
```

*Layer 1 · runtime* · **Aliases:** `queue.len`

Returns the number of elements in the collection.

### `new`

```ori
() -> queue.Queue<_#0>
```

*Layer 1 · runtime* · **Aliases:** `queue.new`

Creates a new empty instance of the queue collection.

### `peek`

```ori
(arg0: queue.Queue<_#0>) -> optional<_#0>
```

*Layer 1 · runtime* · **Aliases:** `queue.peek`

Returns the highest-priority element without removing it.

### `to_list`

```ori
(arg0: queue.Queue<_#0>) -> list<_#0>
```

*Layer 1 · runtime* · **Aliases:** `queue.to_list`

Converts the collection into a plain list.
