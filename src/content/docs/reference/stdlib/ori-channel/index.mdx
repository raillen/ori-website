Channel-based message passing for concurrent communication between tasks.

### `close`

```ori
(arg0: channel.Channel<_#0>) -> void
```

*Layer 1 · runtime* · **Aliases:** `channel.close`

Closes an open file handle, releasing the associated resources.

### `create`

```ori
() -> channel.Channel<_#0>
```

*Layer 1 · runtime* · **Aliases:** `channel.create`

Creates a new channel for message passing between tasks.

### `receive`

```ori
(arg0: channel.Channel<_#0>) -> result<_#0, channel.ReceiveError>
```

*Layer 1 · runtime* · **Aliases:** `channel.receive`

Receives a value from the channel, blocking until one is available.

### `send`

```ori
(arg0: channel.Channel<_#0>, arg1: _#0) -> result<void, channel.SendError>
```

*Layer 1 · runtime* · **Aliases:** `channel.send`

Sends a value through the channel.
