JSON (JavaScript Object Notation) parser and encoder. Allows serializing Ori values into strings and deserializing JSON payloads back to language structures.

### `parse`

```ori
(arg0: string) -> result<<def DefId(9999)>, string>
```

*Layer 1 · runtime* · **Aliases:** `json.parse`

Parses a JSON string into an Ori value.

### `stringify`

```ori
(arg0: <def DefId(9999)>) -> string
```

*Layer 1 · runtime* · **Aliases:** `json.stringify`

Serializes a value to a compact JSON string.

### `stringify_pretty`

```ori
(arg0: <def DefId(9999)>) -> string
```

*Layer 1 · runtime* · **Aliases:** `json.stringify_pretty`

Serializes a value to a pretty-printed JSON string with indentation.
