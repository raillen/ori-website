Singly-linked list data structure. Supports O(1) insertion at head, cursor-based traversal, and sequential access patterns.

### `clear`

```ori
(arg0: linked_list.LinkedList<_#0>) -> void
```

*Layer 1 · runtime* · **Aliases:** `linked_list.clear`

Removes all elements from the collection, resetting it to empty.

*Layer 1 · runtime* · **Aliases:** `linked_list.clone`
