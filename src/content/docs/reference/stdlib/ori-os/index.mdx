Operating system interaction primitives. Provides access to environment variables, process info, platform detection, and directory navigation.

### `arch`

```ori
() -> string
```

*Layer 1 · runtime* · **Aliases:** `os.arch`

Returns the CPU architecture (e.g., x86_64, aarch64).

### `args`

```ori
() -> list<string>
```

*Layer 1 · runtime* · **Aliases:** `os.args`

Returns the command-line arguments passed to the program.

### `change_dir`

```ori
(arg0: string) -> result<void, string>
```

*Layer 1 · runtime* · **Aliases:** `os.change_dir`

Changes the current working directory to the specified path.

### `current_dir`

```ori
() -> result<string, string>
```

*Layer 1 · runtime* · **Aliases:** `os.current_dir`

Returns the current working directory path.

### `env`

```ori
(arg0: string) -> optional<string>
```

*Layer 1 · runtime* · **Aliases:** `os.env`

Returns the value of an environment variable.

### `exit`

```ori
(arg0: int) -> void
```

*Layer 1 · runtime* · **Aliases:** `os.exit`

Terminates the process with the specified exit code.

### `pid`

```ori
() -> int
```

*Layer 1 · runtime* · **Aliases:** `os.pid`

Returns the current process ID.

### `platform`

```ori
() -> string
```

*Layer 1 · runtime* · **Aliases:** `os.platform`

Returns the current operating system platform name.
