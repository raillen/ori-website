Time measurement, clock manipulation, and thread execution suspension routines.

### `duration_ms`

```ori
(arg0: int, arg1: int) -> int
```

*Layer 1 · runtime* · **Aliases:** `time.duration_ms`

### `now`

```ori
() -> int
```

*Layer 1 · runtime* · **Aliases:** `time.now`

Returns the current system timestamp in UNIX epoch seconds or milliseconds.

* **Applicability:** Used for benchmarks, logging date metrics, calculating task durations, and generating unique ID seeds.

```ori
// Example
import ori.time as time
import ori.io as io

func main()
    const start = time.now()
    // run task
    const elapsed = time.now() - start
    io.println("Elapsed: " + string(elapsed) + "s")
end
```

### `sleep`

```ori
(arg0: int) -> void
```

*Layer 1 · runtime* · **Aliases:** `time.sleep`

Suspends execution of the current thread/task for a given duration in milliseconds.

* **Applicability:** Used in loops to set intervals, rate-limiting requests, implementing game cycles, or delays in automation.

```ori
// Example
import ori.time as time
import ori.io as io

func main()
    io.println("Waiting...")
    time.sleep(1000) // Delay 1 second
    io.println("Finished.")
end
```
