### `ori.string.split`

```ori
(arg0: string, arg1: string) -> list<string>
```

- Layer 1 · runtime

Splits a string by a delimiter substring, returning a list of segments.

**Applicability:**
Useful for CSV processing, parsing command arguments, and text segment extraction.

```ori
import ori.string as str
import ori.io as io

func main()
    const items = str.split("apple,banana,cherry", ",")
    // items contains ["apple", "banana", "cherry"]
end
```
