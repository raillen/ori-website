String utility operations. Provides length checking, splitting, search filters, case modifications, and conversions to raw byte arrays.

### `chars`

```ori
(arg0: string) -> list<string>
```

*Layer 1 · runtime*

Returns the individual characters of the string as a list.

### `concat`

```ori
(arg0: string, arg1: string) -> string
```

*Layer 1 · runtime*

Concatenates two strings into one.

### `contains`

```ori
(arg0: string, arg1: string) -> bool
```

*Layer 1 · runtime*

Returns true if the collection contains the specified element.

### `ends_with`

```ori
(arg0: string, arg1: string) -> bool
```

*Layer 1 · runtime*

Returns true if the string ends with the given suffix.

### `from_bytes`

```ori
(arg0: bytes) -> result<string, string>
```

*Layer 1 · runtime* · **Aliases:** `string.from_bytes`

Converts a byte buffer to a string, interpreting bytes as UTF-8.

### `index_of`

```ori
(arg0: string, arg1: string) -> int
```

*Layer 1 · runtime* · **Aliases:** `string.index_of`

Returns the index of the first occurrence of the given element, or -1 if not found.

### `join`

```ori
(arg0: list<string>, arg1: string) -> string
```

*Layer 1 · runtime* · **Aliases:** `string.join`

Joins a list of strings with the given separator.

### `len`

```ori
(arg0: string) -> int
```

*Layer 1 · runtime*

Returns the number of elements in the collection.

### `pad_left`

```ori
(arg0: string, arg1: int, arg2: string) -> string
```

*Layer 1 · runtime* · **Aliases:** `string.pad_left`

Pads the string on the left to the target length with the given character.

### `pad_right`

```ori
(arg0: string, arg1: int, arg2: string) -> string
```

*Layer 1 · runtime* · **Aliases:** `string.pad_right`

Pads the string on the right to the target length with the given character.

### `parse_float`

```ori
(arg0: string) -> result<float, string>
```

*Layer 1 · runtime* · **Aliases:** `string.parse_float`

Parses the string as a floating-point number.

### `parse_int`

```ori
(arg0: string) -> result<int, string>
```

*Layer 1 · runtime* · **Aliases:** `string.parse_int`

Parses the string as an integer.

### `repeat`

```ori
(arg0: string, arg1: int) -> string
```

*Layer 1 · runtime* · **Aliases:** `string.repeat`

Returns a new string formed by repeating the string N times.

### `replace`

```ori
(arg0: string, arg1: string, arg2: string) -> string
```

*Layer 1 · runtime*

Replaces the first occurrence of a substring with a new value.

### `slice`

```ori
(arg0: string, arg1: int, arg2: int) -> string
```

*Layer 1 · runtime*

Returns a new collection containing elements from the specified start to end index.

### `split`

```ori
(arg0: string, arg1: string) -> list<string>
```

*Layer 1 · runtime*

Splits a string by a delimiter substring, returning a list of segments.

* **Applicability:** Useful for CSV processing, parsing command arguments, and text segment extraction.

```ori
// Example
import ori.string as str
import ori.io as io

func main()
    const items = str.split("apple,banana,cherry", ",")
    // items contains ["apple", "banana", "cherry"]
end
```
