LIFO stack data structure. Supports push, pop, and peek operations.

### `clear`

```ori
(arg0: stack.Stack<_#0>) -> void
```

*Layer 1 · runtime* · **Aliases:** `stack.clear`

Removes all elements from the collection, resetting it to empty.

### `clone`

```ori
(arg0: stack.Stack<_#0>) -> stack.Stack<_#0>
```

*Layer 1 · runtime* · **Aliases:** `stack.clone`

Returns a deep copy of the collection, independent of the original.

### `is_empty`

```ori
(arg0: stack.Stack<_#0>) -> bool
```

*Layer 1 · runtime* · **Aliases:** `stack.is_empty`

Returns true if the collection contains no elements.

### `len`

```ori
(arg0: stack.Stack<_#0>) -> int
```

*Layer 1 · runtime* · **Aliases:** `stack.len`

Returns the number of elements in the collection.

### `new`

```ori
() -> stack.Stack<_#0>
```

*Layer 1 · runtime* · **Aliases:** `stack.new`

Creates a new empty instance of the stack collection.

### `peek`

```ori
(arg0: stack.Stack<_#0>) -> optional<_#0>
```

*Layer 1 · runtime* · **Aliases:** `stack.peek`

Returns the highest-priority element without removing it.

### `pop`

```ori
(arg0: stack.Stack<_#0>) -> optional<_#0>
```

*Layer 1 · runtime* · **Aliases:** `stack.pop`

Removes and returns the last element from the collection.

### `push`

```ori
(arg0: stack.Stack<_#0>, arg1: _#0) -> void
```

*Layer 1 · runtime* · **Aliases:** `stack.push`

Adds an element to the collection.

### `to_list`

```ori
(arg0: stack.Stack<_#0>) -> list<_#0>
```

*Layer 1 · runtime* · **Aliases:** `stack.to_list`

Converts the collection into a plain list.
