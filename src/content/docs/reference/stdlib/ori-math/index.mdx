General mathematical constants and helper routines. Contains basic arithmetic bounds checking, rounding, trigonometry, and logarithms.

### `abs`

```ori
(arg0: int) -> int
```

*Layer 1 · runtime*

Returns the absolute value of the number.

### `acos`

```ori
(arg0: float) -> float
```

*Layer 1 · runtime* · **Aliases:** `math.acos`

Returns the arc cosine (inverse cosine) of the value.

### `asin`

```ori
(arg0: float) -> float
```

*Layer 1 · runtime* · **Aliases:** `math.asin`

Returns the arc sine (inverse sine) of the value.

### `atan`

```ori
(arg0: float) -> float
```

*Layer 1 · runtime* · **Aliases:** `math.atan`

Returns the arc tangent (inverse tangent) of the value.

### `atan2`

```ori
(arg0: float, arg1: float) -> float
```

*Layer 1 · runtime* · **Aliases:** `math.atan2`

Returns the arc tangent of y/x using the signs of both arguments.
