System process execution. Provides functions to run shell commands and capture their output.

### `run`

```ori
(arg0: string, arg1: list<string>) -> result<int, string>
```

*Layer 1 · runtime* · **Aliases:** `process.run`

Executes a system command and waits for it to complete.

### `run_capture`

```ori
(arg0: string, arg1: list<string>) -> result<map<string, string>, string>
```

*Layer 1 · runtime* · **Aliases:** `process.run_capture`

Executes a system command and captures its stdout and stderr output.
