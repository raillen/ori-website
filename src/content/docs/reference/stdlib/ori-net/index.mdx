Networking module. Provides interfaces for launching TCP/UDP servers and establishing network connection streams.

### `close`

```ori
(arg0: net.Connection) -> void
```

*Layer 1 · runtime* · **Aliases:** `net.close`

Closes an open file handle, releasing the associated resources.

### `connect`

```ori
(arg0: string, arg1: int, arg2: int) -> result<net.Connection, string>
```

*Layer 1 · runtime* · **Aliases:** `net.connect`

Establishes a TCP connection to the specified host and port.

### `is_closed`

```ori
(arg0: net.Connection) -> bool
```

*Layer 1 · runtime* · **Aliases:** `net.is_closed`

Returns true if the network connection has been closed.

### `read_some`

```ori
(arg0: net.Connection, arg1: int) -> result<bytes, string>
```

*Layer 1 · runtime* · **Aliases:** `net.read_some`

Reads some bytes from the network stream.

### `write_all`

```ori
(arg0: net.Connection, arg1: bytes) -> result<void, string>
```

*Layer 1 · runtime* · **Aliases:** `net.write_all`

Writes all bytes to the network stream.
