### `print`

```ori
(arg0: string) -> void
```

*Layer 1 · runtime*

Writes a string to the standard output without appending a newline.

* **Applicability:** Used for printing prompt messages, building progress indicators, or custom formatted output.

```ori
// Example
import ori.io as io

func main()
    io.print("Enter value: ")
end
```

### `println`

```ori
(arg0: string) -> void
```

*Layer 1 · runtime*

Writes a string to the standard output, followed by a newline.

* **Applicability:** Used for printing log messages, general command line outputs, and standard debugging text.

```ori
// Example
import ori.io as io

func main()
    io.println("Application started successfully.")
end
```
