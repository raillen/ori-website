Test assertion primitives. Provides functions for comparing values, checking allocations, and managing test execution.

### `assert`

```ori
(arg0: bool, arg1: string) -> void
```

*Layer 1 · runtime* · **Aliases:** `test.assert`

Asserts that a condition is true. Fails the test if false.

### `assert_eq`

```ori
(arg0: _#0, arg1: _#0) -> void
```

*Layer 1 · runtime* · **Aliases:** `test.assert_eq`

Asserts that two values are equal. Fails the test if they differ.

### `assert_ne`

```ori
(arg0: _#0, arg1: _#0) -> void
```

*Layer 1 · runtime* · **Aliases:** `test.assert_ne`

Asserts that two values are not equal. Fails the test if they are equal.

### `assert_no_leaks`

```ori
(arg0: string) -> int
```

*Layer 1 · runtime* · **Aliases:** `test.assert_no_leaks`

Asserts that no memory leaks occurred during the test.

### `collect_cycles`

```ori
() -> int
```

*Layer 1 · runtime* · **Aliases:** `test.collect_cycles`

Triggers cycle collection and returns the number of cycles collected.

### `fail`

```ori
(arg0: string) -> never
```

*Layer 1 · runtime* · **Aliases:** `test.fail`

Immediately fails the current test with a message.

### `live_allocations`

```ori
() -> int
```

*Layer 1 · runtime* · **Aliases:** `test.live_allocations`

Returns the number of currently live memory allocations.

### `skip`

```ori
(arg0: string) -> void
```

*Layer 1 · runtime* · **Aliases:** `test.skip`

Skips the current test.
