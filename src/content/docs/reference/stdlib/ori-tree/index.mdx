Tree data structure. Supports hierarchical node management with pre-order, post-order, and breadth-first traversals.

### `add_child`

```ori
(arg0: tree.Tree<_#0>, arg1: tree.NodeId, arg2: _#0) -> tree.NodeId
```

*Layer 1 · runtime* · **Aliases:** `tree.add_child`

Adds a child node to the specified parent node.

### `breadth_first`

```ori
(arg0: tree.Tree<_#0>) -> list<tree.NodeId>
```

*Layer 1 · runtime* · **Aliases:** `tree.breadth_first`

Traverses the tree level by level and returns node values.

### `children`

```ori
(arg0: tree.Tree<_#0>, arg1: tree.NodeId) -> list<tree.NodeId>
```

*Layer 1 · runtime* · **Aliases:** `tree.children`

Returns the direct children of the specified node.

### `clone`

```ori
(arg0: tree.Tree<_#0>) -> tree.Tree<_#0>
```

*Layer 1 · runtime* · **Aliases:** `tree.clone`

Returns a deep copy of the collection, independent of the original.

### `clone_subtree`

```ori
(arg0: tree.Tree<_#0>, arg1: tree.NodeId) -> tree.Tree<_#0>
```

*Layer 1 · runtime* · **Aliases:** `tree.clone_subtree`
