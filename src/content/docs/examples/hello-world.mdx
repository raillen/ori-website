---
title: Hello world
description: Minimal Ori program with namespace, import, and main.
---

```ori
namespace app.main

import ori.io as io

func main()
    io.println("Hello, world!")
end
```

Run with `ori run main.orl` or `cargo run -p ori-driver -- run main.orl` from the compiler workspace.

## Why this example is the baseline

- The namespace is explicit.
- The import uses the common `io` alias.
- `main` has one job.
- The output line is simple enough for a first check.

For larger examples, follow the [cookbook standards](/cookbook/).
