---
title: Planejador de rotas
description: Modele rotas com graph.Graph e use graph.algorithms para consultar caminhos.
---

Objetivo: representar lugares e conexões.

Este projeto usa grafo em vez de listas soltas.

## Mapa

```ori
namespace routes.map

import ori.graph as graph
import ori.graph.algorithms as graph_algorithms

public func build_city() -> graph.Graph<string>
    const city: graph.Graph<string> = graph.new<string>()

    graph.add_node(city, "home")
    graph.add_node(city, "market")
    graph.add_node(city, "library")

    graph.add_edge(city, "home", "market")
    graph.add_edge(city, "market", "library")

    return city
end

public func can_reach(city: graph.Graph<string>, from: string, to: string) -> bool
    return graph_algorithms.has_path(city, from, to)
end
```

## Main

```ori
namespace routes.main

import ori.io as io
import routes.map as routes

func main()
    const city = routes.build_city()
    if routes.can_reach(city, "home", "library")
        io.print("route found")
    else
        io.print("no route")
    end
end
```

## O que este projeto ensina

- Grafo modela relações entre nós.
- `has_path` responde uma pergunta de alcance.
- O domínio fica mais claro do que pares soltos de strings.
