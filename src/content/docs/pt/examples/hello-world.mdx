---
title: Hello World
description: Programa mínimo em Ori contendo namespace, import e função main.
---

```ori
namespace app.main

import ori.io as io

func main()
    io.println("Olá, mundo!")
end
```

Para rodar este exemplo, execute o comando `ori run main.orl` no terminal, ou `cargo run -p ori-driver -- run main.orl` a partir da raiz do workspace do compilador.

## Por que este exemplo é a base

- O namespace é explícito.
- O import usa o alias comum `io`.
- `main` tem um único trabalho.
- A linha de saída é simples para uma primeira verificação.
