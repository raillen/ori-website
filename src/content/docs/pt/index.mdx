## Por que Ori?

Ori prioriza **código legível** com anotações de tipo obrigatórias, `result` / `optional` para erros, e backend nativo Cranelift.

- [Primeiros passos](/pt/getting-started/)
- [Referência (EN)](/reference/) — gerada a partir do compilador
- [Erros, null, void](/pt/language/errors-null-void/)

```ori
namespace app.main
import ori.io as io
func main()
    io.println("Olá, Ori!")
end
```
